# 🍎 Vaishali Exotic Fruits - Zepto Vendor Onboarding Portal

A modern, responsive vendor onboarding website for Zepto delivery platform, showcasing Vaishali Exotic Fruits store and products for verification purposes.

## 📋 Features

✅ **Modern, Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
✅ **Zepto Branding** - Official color scheme and professional layout
✅ **Product Showcase** - Display all exotic fruits with pricing and details
✅ **Store Information** - Location, hours, contact details, and ratings
✅ **Benefits Section** - Highlight advantages of selling on Zepto
✅ **Performance Stats** - Show customer ratings and order metrics
✅ **Onboarding Steps** - Clear 4-step verification process
✅ **Smooth Animations** - Engaging scroll effects and interactions
✅ **Mobile Optimized** - Fast loading and touch-friendly interface
